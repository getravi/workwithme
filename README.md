# Work With Me

Work With Me is an open-source autonomous coworker and chat application built with Tauri, React, and TypeScript. It features a rich Markdown-enabled chat interface and is designed to act as an expert autonomous software engineer working directly alongside you.

## Features

- **Tauri + React + TypeScript** stack for a fast, native desktop experience.
- **Rich Markdown Chat**: Support for GitHub Flavored Markdown, syntax highlighting, and custom rendering.
- **Agentic Capabilities**: Designed to integrate with local Model Context Protocol (MCP) tools and subagents.
- **Proactive & Autonomous**: Built with principles of proactive problem solving and meaningful actions.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://www.rust-lang.org/tools/install)
- [Tauri Prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites)

## Installation and Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/workwithme.git
   cd workwithme
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run tauri dev
   ```

4. **Build for production:**
   ```bash
   npm run tauri build
   ```
