# Credits

Work With Me is built on the shoulders of an incredible ecosystem of open-source projects. We are deeply grateful to the authors and contributors of the following:

## Core Engine

### [pi-mono](https://github.com/badlogic/pi-mono) — Mario Zechner (badlogic)

The heart of Work With Me. `pi-mono` provides the agent runtime (`@mariozechner/pi-coding-agent`) and the unified multi-provider LLM API (`@mariozechner/pi-ai`) that powers every conversation and tool call in the app.

## Bundled Extensions

Work With Me ships with the following community-built pi-extensions:

| Project | Author | What it does |
|---------|--------|--------------|
| [glimpse](https://github.com/HazAT/glimpse) | HazAT | UI panels and visual tooling |
| [pi-mcp-adapter](https://github.com/nicobailon/pi-mcp-adapter) | nicobailon | Connect to any MCP (Model Context Protocol) tool server |
| [pi-subagents](https://github.com/nicobailon/pi-subagents) | nicobailon | Spawn and coordinate parallel subagents |
| [pi-smart-sessions](https://github.com/HazAT/pi-smart-sessions) | HazAT | Intelligent session continuity and management |
| [pi-parallel](https://github.com/HazAT/pi-parallel) | HazAT | Run tasks in parallel |
| [chrome-cdp-skill](https://github.com/pasky/chrome-cdp-skill) | pasky | Browser automation via Chrome DevTools Protocol |

## Desktop Framework

### [Tauri](https://tauri.app) — Tauri contributors

The native desktop shell. Tauri makes it possible to ship Work With Me as a fast, lightweight native app on macOS, Windows, and Linux.

---

## Inspiration

Work With Me was inspired by the idea behind Claude Cowork — the concept of a persistent, autonomous coding partner that works alongside you in your environment. This project exists to bring that experience to everyone, openly, with the freedom to choose any LLM.
